## FTorch Community Resources

On these pages you can see the latest news from the FTorch development team, past
presentations and workshops, some examples of use cases and publications
that build on and cite FTorch, and a copy of the [changelog](|page|/community/changelog.html)
detailing any changes for each release and the latest code.

To ensure that you remain up to date with the latest community developments you can
[subscribe to the FTorch mailing list](https://www.jiscmail.ac.uk/cgi-bin/webadmin?A0=ftorch-announce)
hosted on JiscMail.
